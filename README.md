# Simple Editor Calculator for Obsidian

![demo of functionality](https://github.com/SomeNoY/obsidian-simple-editor-calculator/blob/master/assets/demo.gif)

Very simple plugin that can calculate expressions in editor.

## How to use

1. Select your numerical expression
2. Open Command palette (<kbd>Ctrl+ P</kbd> by default) and search for "Calculator"
3. Choose your command and press <kbd>Enter</kbd>

You also can bind any command in Obsidian settings.

## Commands

- **Calculate** - calculates expression and formmats to `mono`
- **Calculate without formatting** - doing the same as **Calculate** but without `mono` formatting
- **Calculate with replacement** - replaces expression with result
